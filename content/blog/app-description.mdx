# このWebアプリについて

こんにちは、Kairi Uenoです。本日は私が開発した「Blog Writer」というWebアプリケーションについて、その概要と特徴をご紹介します。

## Blog Writerとは

Blog Writerは、ブログ投稿アプリです。ユーザーは簡単に記事を投稿し、ダッシュボードから管理することができます。また、使いやすさとシンプルなデザインを重視しているため、どなたでも直感的に操作できるように設計されています。

皆さんもぜひログインして、dashboardページにある「sample」という記事を参考にしながら、記事を作成してみてください。操作を通じて、Blog Writerがどのように機能するかを体感していただけると思います。

## 主な特徴

Blog Writerには、以下のような特徴があります：

1. **レスポンシブデザイン**: スマートフォン、PCなど、さまざまな画面サイズに対応しています。

2. **簡単な記事投稿**: 記事の作成から公開までのプロセスがシンプルで、技術的な知識がなくても操作可能です。

3. **ダッシュボード機能**: ユーザー専用のダッシュボードで、投稿した記事を一覧管理することができます。編集や削除も簡単に行えます。

4. **モダンなUI/UX**: シンプルで直感的なインターフェースにより、ストレスなく操作できます。

## 使用技術

Blog Writerは、モダンなWeb開発技術を駆使して構築されています。以下は、開発に使用した主な技術スタックです：

- フロントエンド: React, Next.js
- スタイリング: Tailwind CSS
- バックエンド: Node.js, Supabase
- データベース: Supabase

## 今後の展望

Blog Writerは、現時点では最低限の機能しか提供しておらず、今後さらに改善を続けていく予定です。特に、以下の機能追加を計画しています：

- Webアップロード機能: Notionのように、作成した記事をWeb上にアップロードできる機能を導入します。

- カスタマイズ可能なテーマ: ユーザーがブログ記事の外観やデザインを自由にカスタマイズできるテーマオプションを提供します。

- 検索機能: 投稿された記事を簡単に検索できるフィルター機能を実装します。

## さいごに

Blog Writerは、私自身の開発スキルの向上を目指して作成したプロジェクトですが、他のクリエイターや開発者の方々にも役立つアプリケーションとなることを願っています。このアプリを通じて、皆さんが自身のプロジェクトや成果を効果的に発信できるようお手伝いできれば幸いです。引き続き、Blog Writerの発展にご期待ください。
