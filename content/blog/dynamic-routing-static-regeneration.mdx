# このWebアプリについて

こんにちは、kairi Uenoです。本日は私が開発した「Port folio」というWebアプリケーションについて、その概要と特徴をご紹介します。

## Port folioとは

Port folioは、クリエイターやエンジニアが自身の作品やスキルを効果的に紹介するためのポートフォリオ作成ツールです。簡単な操作で、プロフェッショナルな印象を与えるポートフォリオサイトを構築することができます。

## 主な特徴

1. **レスポンシブデザイン**: どのデバイスでも最適な表示が可能です。
2. **カスタマイズ性**: 豊富なテンプレートと色のオプションがあります。
3. **SEO対策**: 検索エンジンに優しい構造になっています。
4. **プロジェクト管理**: 作品を簡単に追加・編集できます。

## 使用技術

Port folioは以下の技術を使用して開発されています：

- フロントエンド: React, Next.js
- スタイリング: Tailwind CSS
- バックエンド: Node.js, Express
- データベース: MongoDB
